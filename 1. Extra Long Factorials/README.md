# Extra Long Factorials

[![]( https://img.shields.io/badge/Бодлогын_линк-blue)](https://www.hackerrank.com/challenges/extra-long-factorials/problem?isFullScreen=true)

## Бодлогын тайлбар

Бүхэл тооны факториал (n!) гэдэг нь:

n!=n×(n−1)×(n−2)×⋯×3×2×1
Өгөгдсөн бүхэл тооны факториалыг тооцоолж хэвлэнэ.

Жишээ нь, хэрэв 

n=30 бол 

30×29×28×⋯×2×1 үржвэрийг тооцоолж:

265252859812191058636308480000000 гэсэн хариуг авна.

#### Функцийн тодорхойлолт
extraLongFactorials функцийг доорх дагуу гүйцэтгэнэ. Үр дүнг хэвлэх бөгөөд буцаах шаардлагагүй.

Функц нь дараах параметртэй:

- n: бүхэл тоо

#### Анхаар: 
n>20 үед факториалын утгыг энгийн 64-бит (long) хувьсагчид хадгалж болохгүй. Ийм тохиолдолд Java, Python, Ruby зэрэг хэлнүүдэд BigInteger төрөл ашиглах шаардлагатай.

#### Оролтын формат
Ганц бүхэл тоо 
n

#### Хязгаарлалт

1≤n≤100

#### Гаралтын формат
n-ийн факториалыг хэвлэнэ.

#### Жишээ оролт
25

#### Жишээ гаралт
15511210043330985984000000

#### Тайлбар
25!=25×24×23×⋯×3×2×1

## Бодолт

факториалаас гарсан маш урт тоог хадгалахын тулд BigInteger классыг ашигласан ба факториалыг олохдоо 1-ээс эхлүүлэн n хүртэл давталт хийж, давталт бүр дээр 1 анхны утгатай үр дүнг хадгалах result-ийг тухайн тоогоор үржинэ.

```bash
        BigInteger result = BigInteger.ONE;
    
        for (int i = 1; i <= n; i++) {
            result = result.multiply(BigInteger.valueOf(i));
        }
        System.out.println(result);
```
